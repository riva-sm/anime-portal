# Учебный проект Glo Academy аниме-портал на JS
## Выполнила: Филиппова Ксения

### Технологии:
- HTML
- SASS 
- Swiper
- JavaScript

Реализованы:
- Категории
- Фильтр аниме по жанру
- Сортировка аниме по количеству просмотров
- Сортировка аниме по жанру, тегам
- Слайдер
- Динамически сменяющиеся страницы с аниме
- Все аниме берутся из базы данных в JSON формате (реализовано через firebase)
